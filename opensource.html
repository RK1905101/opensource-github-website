OPEN SOURCE

Open source is is software with source code that anyone can inspect, modify, and enhance and it is available publicly
It is a decentralized software development model which encourages open collaboration.
Peer production with products is the main aim and principle of open-source software development which consist source code, blueprints, and documentation freely available and editable to the public.


Now, coming down to advantages of open source, open source takes less hardware cost, is high quality software, integrated and licenced with lower software cost.
It can also be easily scaled and managed.
It is also very flexible and agile as compared to other software, and also have high security also its future scope is high.
