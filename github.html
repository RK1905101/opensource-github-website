GIT & GITHUB

GitHub is a website and cloud-based service that helps developers store and manage their code, as well as track and control changes to their code.
Git is a specific open-source version control and distributed version control system,which allows for easy branching and merging. Basically, the difference between both of them is that, github is ui based and git is command line based.Infact, both of them are user friendly.
It allows, team or any organisation to work together on a certain project, from anywhere around the world, with security
