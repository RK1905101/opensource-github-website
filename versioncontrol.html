VERSION CONTROL

Version control is the practice of tracking and managing changes to software code.
It is a kind of tool which helps software team to manage changes to source code over time.
It keeps track of each and every modification to the code in a kind of database and compare earlier versions of the code to help fix the mistake while minimizing disruption.
Its advatanges are to keep track of complete history of the software with tracebility and have the ability to compare and merge the changes with the older version too.
